## 🎯 **HIZLI ERİŞİM**

### **Visual Studio Code'da:**
1. VS Code'u açın
2. GitHub Copilot extension'ın aktif olduğundan emin olun
3. Herhangi bir dosyayı açın
4. Bana şöyle yazın:

```
Merhaba GitHub Copilot!
WOLTAXI projemde kaldığımız yerden devam etmek istiyorum.
```
